===== API and AUTH DESIGN

“Instead of multiple guards, I used a single authentication guard with role-based middleware for flexible and scalable access control.”

Route::middleware(['auth:sanctum'])->group(function () {

    // viewer allowed
    Route::get('/dashboard', ...);

    // analyst + admin
    Route::middleware('role:analyst,admin')->group(function () {
        Route::get('/records', ...);
    });

    // admin only
    Route::middleware('role:admin')->group(function () {
        Route::post('/records', ...);
        Route::delete('/records/{id}', ...);
    });
});

I used sanctum  auth automatically installed when run command  "php artisan install:api"
